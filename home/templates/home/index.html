How to use

  • To add a card or set to your account:
    1. Select Collections in the navbar
    2. Select game your collecting
    3. Select "View All Sets"
    4. Search for the set you want to add the card from
    5. Search for the card using the searchbar, make sure only a single result appears in the table, the best way to get this would be that games unique identifier, e.g. for mtg it is "Collectors Number"
    6. When a single result is in the table, use one of the following keys to add or remove a card:
      • +: Add a normal card
      • -: Remove a card
      • *: Add a foil card
      • /: Remove a foil card
    7. Press Save
  • All sets that you own atleast one card from will appear in the your sets page
